# Ticket App

## Cara Menjalankan Aplikasi

1. **Kloning Repository**
```bash
git clone <repository-url>
cd <repository-directory>
```

2. **Jalankan Menggunakan Makefile**
* Migrasi Database
```bash
make migrate-up
```

* Menjalankan Aplikasi
```bash
make run
```
